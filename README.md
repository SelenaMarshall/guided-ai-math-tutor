# Guided AI Tutor (Math for the Trades)

## 📘 Overview

The Guided AI Tutor is an interactive learning application designed to help students master foundational math skills needed in the trades, including fractions and decimals.

This project combines step-by-step problem solving, AI assistance, and visual learning support to create a tutor-like experience.

---

## ❗ Problem

Many learners struggle with basic math skills such as:

* Fractions
* Decimals
* Rounding
* Place value

These skills are critical for success in trade careers (construction, electrical, plumbing, etc.), but are often not taught in a practical, engaging way.

---

## 💡 Solution

This application provides:

* Guided step-by-step explanations
* Practice problems
* Immediate feedback
* AI-powered tutoring support

It helps learners build confidence and accuracy in essential math skills.

---

## ✨ Features

* Fraction operations (add, subtract, multiply, divide)
* Decimal operations
* Rounding support (tenths, hundredths, thousandths)
* Step-by-step explanations
* AI Tutor assistance (LLM integration)
* Clean and interactive user interface

---

## ▶️ How to Run the App

### Step 1: Clone the repository

```bash
git clone https://github.com/YOUR-USERNAME/guided-ai-tutor.git
cd guided-ai-tutor
```

### Step 2: Install dependencies

```bash
pip install -r requirements.txt
```

### Step 3: Run the app

```bash
streamlit run app.py
```
